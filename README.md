# Meeting Room Scheduler Using Java

## Overview
This project is designed to provide a systematic solution for scheduling meetings in a shared environment, ensuring efficient use of meeting rooms.

## Features
- User registration and authentication
- Meeting room availability check
- Schedule a meeting
- Cancel or modify scheduled meetings
- Notifications and reminders

## Project Structure
```
- src/
  - main/
    - java/
    - resources/
  - test/
- docs/
- lib/
```

## Technology Stack
- Java
- Spring Boot
- MySQL
- HTML, CSS, JavaScript

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/harshraj005/Meeting_Room_Scheduler_Using_Java.git
   ```
2. Navigate to the project directory:
   ```
   cd Meeting_Room_Scheduler_Using_Java
   ```
3. Install dependencies:
   ```
   mvn install
   ```

## Usage
To run the application:
```
   mvn spring-boot:run
```

## Components
- Backend: Java with Spring Boot
- Frontend: HTML, CSS, JavaScript
